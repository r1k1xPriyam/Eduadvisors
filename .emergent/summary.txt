<analysis>**original_problem_statement:**
The user wants to build a comprehensive website for their educational consultancy, Edu Advisor. The project started as a landing page and evolved into a full-stack application with a React frontend and FastAPI backend.

PRODUCT REQUIREMENTS:
1.  **Public Website**: A multi-section landing page with a student query form and a GET COUNSELLING NOW button. The site must support light and dark modes.
2.  **Login Portals**: Secure login for Admin and Consultant roles.
3.  **Admin Dashboard**:
    *   View all student queries, consultant reports, and call statistics.
    *   **Bulk Delete**: Delete data with filters and password confirmation.
    *   **Manage Consultants**: Admins can permanently add, delete, or modify any consultant's credentials.
    *   **View & Delete Call Stats**: View detailed call statistics (successful, failed, attempted) for each consultant by clicking on the numbers. Ability to delete call stats for individual consultants.
    *   **View & Manage Reminders**: Access a list of Next Calling Reminders for all consultants, with the ability to delete them.
    *   **Analytics Dashboard**: View 5 charts: Call Stats Over Time, Reports by Consultant, Admissions by Consultant, Call Status Distribution, and Top 5 Competitive Exam Preferences.
4.  **Consultant Portal**:
    *   Submit detailed student calling reports. Mandatory fields must be enforced with warnings.
    *   Handle duplicate reports (based on phone number) by overwriting the old report after a confirmation prompt.
    *   **Quick Call Logging**: A simple form to log only Failed or Attempted calls. Contact number is mandatory.
    *   **View Data**: View own submitted reports, admissions, and call statistics (summary and detailed).
    *   **Next Calling Reminder**: An optional calendar input to set a follow-up reminder.
    *   **Reminder Management**: A dedicated section displays all upcoming reminders, sorted by date. Consultants receive a notification on login for due reminders and can mark them as Already followed up or Ignore.
    *   **Bulk Report Upload**: Ability to download a sample CSV, fill it, and upload it to submit multiple reports. A warning prompt should appear before uploading.
